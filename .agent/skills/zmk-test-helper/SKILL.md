# ZMK Test Helper — テスト・検証支援

## スキル読み込み通知

このスキルが読み込まれたら、必ず以下の通知をユーザーに表示してください：

> 🧪 **ZMK Test Helper スキルを読み込みました**  
> `west test` の実行やテスト結果の解析をサポートします。

---

## 概要

ZMK (Zephyr) のテストフレームワークを使用したユニットテストの実行と、その結果に基づくデバッグを支援します。
ビルドが通った後の「論理的な挙動の正しさ」を検証することに特化しています。

---

## 使用タイミング

- 新しい behavior やマクロを実装し、その挙動をシミュレーションで確認したいとき
- `west test` が失敗し、どのテストケースが落ちているか特定したいとき
- GitHub Actions のテストステップが失敗したとき

---

## 手順

### Step 1: テスト環境の確認

リポジトリ内に `tests/` ディレクトリが存在し、有効なテストケースがあるか確認します。

### Step 2: テストの実行

ローカル環境またはコンテナ環境でテストを実行します。

```bash
# 全てのテストを実行
west test

# 特定のテストケースのみ実行
west test -t <test-name>
```

### Step 3: 結果の解析

出力ログから `FAIL` となっている箇所を特定し、原因（アサーション失敗、タイムアウトなど）を分析します。

### Step 4: 修正と再テスト

原因を特定したらコードを修正し、再度テストを実行して `PASS` になることを確認します。
