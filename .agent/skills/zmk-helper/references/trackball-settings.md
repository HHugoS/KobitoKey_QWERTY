# トラックボール設定リファレンス (PMW3610等)

ZMKで光学センサー（特に PMW3610）を使用する際の推奨設定値をまとめます。

## 基本設定 (DPI & サンプリングレート)

感度（DPI）とサンプリングレートは、`.dtsi` などのデバイスツリーファイルで設定します。

```devicetree
&trackball {
    status = "okay";
    // 感度 (DPI): 1-16 (1 = 400dpi, 4 = 1600dpi ...)
    // PMW3610の場合、4-8 程度が一般的
    sensitivity = <4>; 
    
    // サンプリングレート: ms単位
    sampling-rate = <10>; 
};
```

## スクロール動作の調整

トラックボールをスクロールホイールとして使用する場合の設定例。

### スクロールモードの感度
スクロールの「重さ」はマウスエミュレーション側で調整します。

```devicetree
/* スクロールの乗数設定 */
#define SCROLL_DIVIDER 10
```

## 実装パターン

### 1. スナイパーモード（低感度モード）
精密な操作が必要なときに感度を下げる例。レイヤーやホールド・タップ behavior と組み合わせて使用します。

### 2. オートマウスレイヤー
トラックボールを動かした瞬間にマウス操作レイヤーへ自動切り替えする設定（ `zmk,input-listener` ）。

> [!TIP]
> 感度が良すぎて意図せずレイヤーが変わる場合は、`input-processor-wait` を追加してチャタリングを防ぎます。
